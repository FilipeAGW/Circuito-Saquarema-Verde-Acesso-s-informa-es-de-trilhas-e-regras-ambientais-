MVP Definido - Plataforma de informações. 
Entregar uma plataforma simples e funcional onde visitantes possam consultar rapidamente informações atualizadas sobre trilhas e normas ambientais de Saquarema, enquanto administradores têm um painel seguro para gerir essas informações.
Com o objetivo de:
facilidade de uso
velocidade de acesso
clareza das regras ambientais
e eficiencia na gestão de trilhas


Funcionalidades:
Para visitantes, Navegar pelas trilhas disponíveis, Ver detalhes (dificuldade, localização, mapas simples), Ler regras ambientais obrigatórias, Ver status: aberta, fechada, restrição parcial

Para administrador, Login seguro, Gerenciar trilhas (criar, editar, remover), Atualizar regras ambientais, Atualizar status e temporadas de cada trilha.

Telas: 
Home (lista de trilhas)
Página de detalhes da trilha
Login administrativo
Painel administrativo
